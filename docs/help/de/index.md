# Phylax Dokumentation

Willkommen bei der Phylax-Dokumentation.

Phylax ist eine **lokale, datensouveräne Gesundheitsplattform** als Progressive Web App. Du erstellst und pflegst dein lebendes medizinisches Profil im Browser, verschlüsselt mit deinem Master-Passwort. Kein Backend, keine Cloud, keine Telemetrie.
