# ePA-Import (elektronische Patientenakte)

!!! info "Platzhalter (Iteration 1)"
    Diese Seite ist Teil des Stage-1-Geruests. Vollstaendiger Inhalt folgt in einer spaeteren Iteration.

Befunde aus der elektronischen Patientenakte (ePA) lassen sich KI-gestuetzt in das Phylax-Profil uebernehmen. Phylax interpretiert die Befunde nicht; die KI strukturiert sie nur fuer das lebende Profil.

## Geplante Inhalte dieser Seite

- Was die ePA ist und welche Dokumenttypen Phylax verarbeitet (PDF-Befunde, Arztbriefe, Laborberichte)
- Eigenen API-Key fuer OpenAI oder Anthropic hinterlegen (zero-knowledge, keine eigene Cloud)
- Dokument hochladen und Quelle waehlen
- Vorschau des KI-Vorschlags vor dem Schreiben in die Datenbank
- Quelle der importierten Eintraege: `ai` mit Verweis auf das Quelldokument
- Datenschutzhinweise: Daten gehen direkt vom Browser an den gewaehlten KI-Anbieter, kein Phylax-Backend
