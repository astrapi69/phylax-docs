# Profil importieren (Markdown)

!!! info "Platzhalter (Iteration 1)"
    Diese Seite ist Teil des Stage-1-Geruests. Vollstaendiger Inhalt folgt in einer spaeteren Iteration.

Phylax kann ein bestehendes Markdown-Profil im Phylax-Format importieren. Damit laesst sich ein extern gepflegter Profil-Entwurf in die App uebernehmen, ohne Inhalte abzutippen.

## Geplante Inhalte dieser Seite

- Welches Markdown-Format Phylax versteht (Sektionsstruktur, Beobachtungen, Laborwerte, Supplemente, offene Punkte)
- Profil-Datei vorbereiten und in den Import-Wizard laden
- Was passiert beim Import (Vorschau, Konfliktbehandlung, atomarer Schreibvorgang)
- Quelle der importierten Eintraege: `import` (sichtbar pro Eintrag)
- Fehler beim Import lesen und korrigieren
