# Daten exportieren

!!! info "Platzhalter (Iteration 1)"
    Diese Seite ist Teil des Stage-1-Geruests. Vollstaendiger Inhalt folgt in einer spaeteren Iteration.

Phylax bietet mehrere Export-Wege fuer das eigene Profil: Markdown fuer den menschenlesbaren Austausch, verschluesselte `.phylax`-Sicherung fuer den Geraetewechsel, PDF fuer den Arztbesuch.

## Geplante Inhalte dieser Seite

- Markdown-Export: lebendes Profil als lesbare Datei (Phase 2b)
- `.phylax`-Sicherung: verschluesselter Komplett-Export inklusive Master-Schluessel-Material (Phase 6, siehe auch Sicherung)
- PDF-Export: Profil oder Zeitraum zum Mitnehmen (Phase 5, in Vorbereitung)
- Welcher Export wofuer: Austausch, Sicherung, Arzt-Termin
- Datensparsamkeit beim Teilen (kein versehentlicher Versand sensibler Daten)
