# Phylax Documentation

Welcome to the Phylax documentation.

Phylax is a **local-first, data-sovereign health platform** built as a Progressive Web App. You create and maintain your living medical profile in the browser, encrypted with your master password. No backend, no cloud, no telemetry.

## Quick start

- [What is Phylax](getting-started/what-is-phylax.md) - The idea behind the living profile
- [First Steps](getting-started/first-steps.md) - Installation and setup
- [Observations](daily-use/observations.md) - Capture facts, patterns and self-regulation
- [Create a backup](backup/create-and-restore.md) - Back up and restore your data
