# Exporting data

!!! info "Placeholder (Iteration 1)"
    This page is part of the Stage 1 scaffolding. Full content will follow in a later iteration.

Phylax offers several export paths for your profile: Markdown for human-readable exchange, an encrypted `.phylax` backup for moving between devices, and PDF for a physician visit.

## Planned content for this page

- Markdown export: the living profile as a readable file (Phase 2b)
- `.phylax` backup: an encrypted full export including master-key material (Phase 6, see also Backup)
- PDF export: profile or date range to take along (Phase 5, in preparation)
- Which export for which purpose: exchange, backup, physician appointment
- Data minimization when sharing (avoid sending sensitive data accidentally)
